About This Project

Hi, this is a project we built for Cloud Product and Platform Engineering course.

Infrastructure: We deployed a 2-tier Flask application (Flask + MySQL) containerised with Docker and orchestrated using Docker Compose with dedicated volumes and a custom bridge network. The source code is hosted on GitHub, and a Jenkins CI/CD pipeline automates build and deployment on AWS EC2.

ML Model Sentiment Analysis Classifier

The application embeds a lightweight Logistic Regression text classifier trained with scikit-learn. Raw text is vectorised using a Bag-of-Words (CountVectorizer) pipeline and fed into the model to produce a binary sentiment prediction — Positive or Negative. The trained model is serialised with Python's pickle module and loaded at app startup, exposing predictions via the /predict REST endpoint (JSON in / JSON out). The model is retrained inside the Docker image at build time to guarantee sklearn version consistency across all environments.
